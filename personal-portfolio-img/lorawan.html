LoraWAN

A propos

Ce projet vise à développer un système de location de vélos utilisant des badges RFID pour authentifier les abonnements des utilisateurs via le protocole de communication LoraWAN. En intégrant un serveur de réseau TTN pour la communication, une application cloud Node-RED pour le traitement des données, et une base de données InfluxDB pour l'archivage.
